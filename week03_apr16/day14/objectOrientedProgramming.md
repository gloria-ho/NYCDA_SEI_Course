# Object Oriented Programming

## Object v. Variable

```javascript
let myDog = {
	name: 'buzzie',
	age: 3,
	color: 'brown'
};
```

```javascript
let myDogName = 'buzzie';
let myDogAge = 3;
let myDogColor = 'brown';
```
