# Object Oriented Programming

## Object v. Variable

```javascript
let myDog = {
	name: 'buzzie',
	age: 3,
	color: 'brown'
};
```

```javascript
let myDogName = 'buzzie';
let myDogAge = 3;
let myDogColor = 'brown';
```

## Dot Operator & Method

```javascript
let myArray = [2, 4, 6];
myArray.push(8);
myArray.push(10);
myArray.pop();
```
